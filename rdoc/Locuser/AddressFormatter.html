    class Locuser::AddressFormatter

    Class to handle formatting of addresses from a hash containing a specific pattern There is a default class which will format the address from the hash as follows:

    "street1 street2 city, state[not abbreviated]  zip"

    Note that country is left off by default but the #use_country flag will cause this to append to the end. Further the

def format(hsh)
  str = String.new
  unless hsh[:street1] == nil; str << hsh[:street1]; end
  unless hsh[:street2] == nil; str << ((str == nil) ? '' : ' ') << hsh[:street2]; end
  unless hsh[:city] == nil; str << ((str == nil) ? '' : ' ')  << hsh[:city]; end
  unless hsh[:state] == nil; str << ((str == nil) ? '' : ', ')  << hsh[:state]; end
  unless hsh[:zip] == nil; str << ((str == nil) ? '' : '  ')  << hsh[:zip]; end
  unless (!@use_country || hsh[:country] == nil); str << ((str == nil) ? '' : ' ')  << hsh[:country]; end
  return str
end
